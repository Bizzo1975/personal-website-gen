# Windows Scripts Archive

This directory contains Windows-specific batch files (.bat) that were used during Windows development. These scripts have been archived as the project has moved to Ubuntu/Linux development and production.

## Archived Files

- `startup-all.bat` - Windows startup script for development environment
- `backup-full-system.bat` - Windows database backup script
- `run-full-deployment.bat` - Windows deployment orchestrator
- `backup-quick.bat` - Quick backup script for Windows
- `update-projects-to-draft.bat` - Windows utility script

## Linux/Ubuntu Equivalents

All functionality has been recreated in shell scripts (.sh) for Linux/Ubuntu:

- `scripts/deployment/startup-all.sh` - Linux startup script
- `scripts/05-backup-restore/backup-full-system.sh` - Linux database backup script
- `scripts/run-full-deployment.sh` - Linux deployment orchestrator

## Note

These files are kept for reference only. They are not used in the current Ubuntu-based development or production environments.
